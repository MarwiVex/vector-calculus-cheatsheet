# Formulario de Cálculo Vectorial

Este directorio contiene la versión en español del formulario, diseñado en LaTeX. Sirve como un material de referencia ideal para estudiantes, ingenieros e investigadores en matemáticas, física y geometría diferencial. El documento resume de manera concisa las fórmulas esenciales, los teoremas y las metodologías necesarias para el análisis vectorial.

---

## Contenido Principal

* **Álgebra Vectorial y Tensorial:** Productos punto/cruz, notación de índices, Delta de Kronecker, símbolo de Levi-Civita e identidades tensoriales.
* **Geometría Diferencial:** Curvas, longitud de arco, fórmulas de Frenet-Serret, curvatura, torsión y fundamentos de la geometría de superficies (Primera y Segunda Forma Fundamental).
* **Cálculo y Operadores:** Derivadas direccionales, optimización (Hessiana, Lagrange) y el Gradiente, Divergencia, Rotacional y Laplaciano expresados en coordenadas cartesianas, cilíndricas y esféricas.
* **Teoremas Integrales:** Trabajo/Circulación, flujo de superficie, Teorema de Green, Teorema de Stokes y Teorema de la Divergencia de Gauss, junto con el Teorema del Transporte de Reynolds y las Ecuaciones de Maxwell.
* **Intuición Práctica y Metodologías:** Guías para la selección de teoremas, estrategias para sistemas de coordenadas y una sección de fórmulas rápidas para exámenes.

---

## Uso y Compilación

Si deseas compilar el documento PDF directamente desde el código fuente (`calculo_vectorial_cheatsheet.tex`), puedes utilizar tu entorno LaTeX preferido o ejecutar el siguiente comando en tu terminal usando `pdflatex`:

```bash
pdflatex calculo_vectorial_cheatsheet.tex
```

*Nota: Dependiendo de tu configuración, es posible que necesites compilar el documento dos veces para asegurar que todas las referencias cruzadas y el formato se generen correctamente.*
