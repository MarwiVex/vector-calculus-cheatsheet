# Vector Calculus Cheatsheet

This directory contains the English version of the cheatsheet, designed in LaTeX. It serves as an ideal reference material for students, engineers, and researchers in mathematics, physics, and differential geometry. The document concisely summarizes essential formulas, theorems, and workflows necessary for vector analysis.

---

## Core Content

* **Vector & Tensor Algebra:** Dot/Cross products, index notation, Kronecker Delta, Levi-Civita symbol, and tensor identities.
* **Differential Geometry:** Curves, arc length, Frenet-Serret formulas, curvature, torsion, and surface geometry fundamentals (First and Second Fundamental Forms).
* **Calculus & Operators:** Directional derivatives, optimization (Hessian, Lagrange), and the Gradient, Divergence, Curl, and Laplacian expressed across Cartesian, Cylindrical, and Spherical coordinates.
* **Integral Theorems:** Work/Circulation, surface flux, Green's Theorem, Stokes' Theorem, and Gauss's Divergence Theorem, along with the Reynolds Transport Theorem and Maxwell's Equations.
* **Practical Intuition & Workflows:** Theorem selection guides, coordinate system strategies, and a quick-formula section for exams.

---

## Usage and Compilation

If you wish to build the PDF document directly from the source code (`vector_calculus_cheatsheet.tex`), you can use your preferred LaTeX environment or run the following command in your terminal using `pdflatex`:

```bash
pdflatex vector_calculus_cheatsheet.tex
```

*Note: Depending on your setup, you may need to compile the document twice to ensure all cross-references and formatting are generated correctly.*
