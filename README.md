## English Overview
This repository contains a comprehensive cheatsheet for Vector Calculus, designed in LaTeX. It is a unified reference covering everything from basic vector algebra and tensor notation to advanced differential geometry and fundamental integral theorems.

**Main Topics:**
* Tensor Algebra and Index Notation (Einstein Summation, Levi-Civita).
* Coordinate Systems, Transformations, and Differential Geometry (Frenet-Serret).
* Differential Calculus, Optimization, and Differential Operators ($\nabla$) in multiple coordinate systems.
* Integral Theorems (Green, Stokes, Gauss) and Physical Applications (Maxwell's Equations).

## Resumen en Español
Este repositorio contiene un formulario exhaustivo de Cálculo Vectorial diseñado en LaTeX. Es una referencia unificada que abarca desde álgebra vectorial básica y notación tensorial hasta geometría diferencial avanzada y los teoremas integrales fundamentales.

**Contenido Principal:**
* Álgebra Tensorial y Notación de Índices (Suma de Einstein, Levi-Civita).
* Sistemas de Coordenadas, Transformaciones y Geometría Diferencial (Frenet-Serret).
* Cálculo Diferencial, Optimización y Operadores Diferenciales ($\nabla$) en múltiples sistemas de coordenadas.
* Teoremas Integrales (Green, Stokes, Gauss) y Aplicaciones Físicas (Ecuaciones de Maxwell).
